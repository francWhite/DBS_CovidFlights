# DBS_CovidFlights
Databaseproject from Jonathan Winter, Ivan Probst and Francisco Metzger for module DBS at HSLU

## Setup
To create the schmea with all tables execute the following script: ```/scripts/create_shema.sql```
To import the datasets you can either execute the import.py (takes aprox. 25min) or the ```/scripts/import_data.sql``` (takes aprox. 8min) script.
